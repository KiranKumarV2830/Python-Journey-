# 📚 Topics Covered

* Not Equal to (`!=`)
* Greater Than (`>`)
* Less Than (`<`)
* Greater Than or Equal To (`>=`)
* Less Than or Equal To (`<=`)

## Boolean Data Type

* `True`
* `False`

## Conditional Statements

* `if`
* `elif`
* `else`

## Logical Operators

* `and`
* `or`
* `not`

# 💡 Key Takeaways

* Comparison operators always return `True` or `False`.
* `if` executes code only when a condition is `True`.
* `elif` allows checking multiple conditions.
* `else` handles all remaining cases.
* `strip()` removes extra spaces from the beginning and end of a string.
* `lower()` converts text to lowercase, making username comparisons more user-friendly.
