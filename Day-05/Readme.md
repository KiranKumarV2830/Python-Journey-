## 🎤 Interview Preparation

**Q:** What is the difference between `/`, `//`, and `%`?

**Answer:**

* `/` performs normal division and returns a floating-point result.
* `//` performs floor division and returns the whole-number quotient.
* `%` returns the remainder after division.

---

## 📖 Key Takeaways

* Use `/` when an exact decimal answer is required.
* Use `//` when only complete groups or whole numbers are needed.
* Use `%` to find the remainder or check conditions such as even or odd numbers.
* Use the `math` module instead of hardcoding mathematical values like π.
