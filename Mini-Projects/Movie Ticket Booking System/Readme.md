# 🎬 Movie Ticket Booking System (Python Functions)

A beginner-friendly Python mini project that simulates a simple movie ticket booking system using functions and parameters.

---

## 📚 Concepts Practiced

- Python Functions
- Parameters & Arguments
- Function Calling
- Printing Formatted Output
- Program Structure

---

## 📂 Project Features

- 🎥 Display available movie timing
- 🎫 Book movie tickets
- 💰 Calculate ticket bill
- 💳 Select payment method
- 🙏 Display thank you message

---

## 🛠 Functions Used

```python
show_movies()
book_ticket(movie_name, tickets)
calculate_bill(ticket_price, quantity)
payment(method, amount)
thank_you(name)
```
