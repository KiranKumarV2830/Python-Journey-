# 🏧 ATM PIN Verification System

## 📖 Description

A simple ATM authentication system that verifies whether a card is inserted and whether the entered PIN is correct.

---

## ✨ Features

* Card verification
* PIN verification
* Access granted or denied

---

## 🛠 Concepts Used

* Nested `if`
* Logical Operators
* User Input
* Conditional Statements

---

## ▶️ How to Run

```bash
python atm_pin.py
```

---

## 📌 Sample Output

```text
Did you insert the card (Y/N): Y
Enter PIN: 1234

Transaction Started
```
