# 🌡 Temperature Converter

## 📖 Description

A Python program that converts temperatures between Celsius and Fahrenheit.

---

## ✨ Features

* Celsius → Fahrenheit
* Fahrenheit → Celsius
* User input validation
* Rounded output

---

## 🛠 Concepts Used

* Variables
* User Input
* Type Casting
* Arithmetic Operators
* `if-elif-else`
* `round()`

---

## ▶️ How to Run

```bash
python temperature_conversion.py
```

---

## 📌 Sample Output

```text
Select the unit you want to convert to (C/F): C
Enter the temperature: 25

The temperature in Fahrenheit is 77.0°F
```
