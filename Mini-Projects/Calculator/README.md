# 🧮 Calculator

## 📖 Description

A simple Python calculator that performs basic arithmetic operations based on user input.

---

## ✨ Features

* Addition
* Subtraction
* Multiplication
* Division
* Floor Division
* Modulus
* Exponentiation

---

## 🛠 Concepts Used

* Variables
* User Input
* Type Casting
* Arithmetic Operators
* `if-elif-else`
* f-Strings

---

```bash
python calculator.py
```
