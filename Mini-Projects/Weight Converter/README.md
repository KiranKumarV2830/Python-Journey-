# ⚖️ Weight Converter

## 📖 Description

A Python application that converts weight between kilograms and pounds.

---

## ✨ Features

* Kilograms → Pounds
* Pounds → Kilograms
* Rounded output

---

## 🛠 Concepts Used

* Variables
* User Input
* Arithmetic Operators
* `if-elif-else`
* `round()`

---

## ▶️ How to Run

```bash
python weight_converter.py
```

---

## 📌 Sample Output

```text
Enter your weight : 70
Kilograms or Pounds (K/L): K

The weight is 154 Lbs.
```
