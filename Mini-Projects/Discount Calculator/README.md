# 🛍 Discount Calculator

## 📖 Description

A Python program that calculates discounts based on the total purchase amount.

---

## ✨ Features

* 20% discount for purchases above ₹5000
* 10% discount for purchases above ₹2000
* Displays original and final prices

---

## 🛠 Concepts Used

* Variables
* User Input
* Arithmetic Operators
* `if-elif-else`

---

## ▶️ How to Run

```bash
python discount_calculator.py
```

---

## 📌 Sample Output

```text
Enter the purchase amount: 6000

Original Price: ₹6000
Discount: ₹1200
Final Price: ₹4800
```
