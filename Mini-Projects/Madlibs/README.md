# 🎮 Mad Libs Game

## 📖 Description

A fun interactive Python game where the user enters words that are combined into a humorous story.

---

## ✨ Features

* Interactive user input
* Dynamic story generation
* Beginner-friendly project

---

## 🛠 Concepts Used

* Variables
* Strings
* User Input
* f-Strings

---

## ▶️ How to Run

```bash
python madlibs.py
```

---

## 📌 Sample Output

```text
Enter a name: Kiran
Enter an adjective: Amazing
Enter a place: Bengaluru

Kiran is an amazing programmer from Bengaluru!
```

---

## 📚 What I Learned

* Collecting multiple user inputs
* Combining text using f-Strings
* Creating interactive console programs
